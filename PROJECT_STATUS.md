## 🚀 运行状态

### 开发环境要求
- **后端**: JDK 8+, MySQL 8.0+, Redis 6.0+, Maven 3.6+
- **前端**: Node.js 16+, npm/yarn

### 启动流程
1. **数据库**: 导入SQL脚本
2. **后端**: `mvn spring-boot:run` (端口8080)
3. **前端**: `npm run dev` (端口3000)
4. **访问**: http://localhost:3000
